## 1. 图例

| 符号 | 含义 |
|------|------|
| ✅ | **通过** — 满足 PRD 验收标准，可签字 |

**五列说明**

| **文档** | PRD 是否定义功能 ID、用户故事、验收标准 |

## 6. 现场 UAT 脚本（勾选用）

**店名**：____________　**日期**：____________　**店长**：____________

### 6.1 前厅（约 15min）

- [ ] **UAT-T01** 打开首页，5 项 KPI 与当前午市情况基本一致
- [ ] **UAT-T02** 桌态页四态颜色正确，待清桌与现场一致（允许 ≤2 桌误差）
- [ ] **UAT-T03** 翻台建议 Top5 理由可读，领班认同排序
- [ ] **UAT-T04** 5s 内桌态自动更新，无需手动刷新

### 6.2 后厨（约 15min）

- [ ] **UAT-K01** 冷冻/冷藏当前温度与现场温度计一致（±1°C）
- [ ] **UAT-K02** 开门 3min 内收到门磁告警
- [ ] **UAT-K03** 全链路卡片能反映今日收货批次
- [ ] **UAT-K04** 模拟烟雾/燃气事件，critical 出现在告警流

### 6.3 来料 PDA（约 10min）

- [ ] **UAT-P01** 从今日 PO 选一批次（非写死演示项）
- [ ] **UAT-P02** 秤重自动填入，偏差百分比正确
