## 二、差异化三叉戟

```
         看得见 👁️
           ▲
          / \
         /   \
        / 火瞳 \
       /       \
管得着 📊 ─────── 回本快 💰
```

### 👁️ 看得见 — 边缘视觉AI

> 海底捞刚试点，星巴克完全空白。
> 火瞳是唯一覆盖**前厅+后厨双场景**的火锅专用视觉AI。

蒸汽里有锅底、油污下有废料、热雾中识桌态——
通用视觉解决不了的问题，火瞳的火锅场景模型能。

### 📊 管得着 — 层级连锁管控

> Zone→Region→Store→Device。一个总代理管30家加盟店，不需要飞过去。
> 星巴克的管理层级到设备，但不做视觉；海底捞做视觉，但层级只到自家直营。

加盟连锁的世界里，总代最大的痛是什么？
**"我不知道加盟商有没有按规矩来。"**
火瞳就是那个规矩的证明人。

### 💰 回本快 — 算得过来的账

> 2.5个月回本，374%首年ROI。不是科技故事，是算盘。
> 两巨头的方案是百万级军备，火瞳是万元级工具箱。

| | 海底捞系统 | 星巴克系统 | 火瞳 |
|------|:--:|:--:|:--:|
| 单店投入 | 百万级 | 百万级（全球分摊） | 万元级 |
| 回收周期 | 不计回本（自用） | 不计回本（自用） | **2.5月** |
| 适合谁 | 自有1400店 | 自有38000店 | **9万中小连锁** |

### 3.1 四象限定位

```
          技术深度 ▲
                   │
        商汤/旷视  │  ★ 火瞳
        (云端AI·贵)│ (边缘视觉·万元)
                   │
  ─────────────────┼─────────────────→ 商业导向
                   │
    中国电信明厨亮灶│  美团/客如云SaaS
   (免费·政府项目)  │ (云端·不做视觉)
                   │
```

**火瞳是唯一在"高技术深度 × 强商业导向"象限的产品。**

### 3.2 vs 每个对手

| 对手 | 他们打什么 | 火瞳差异化 | 打法 |
|------|-----------|-----------|------|
| **海底捞** | 自研·自用·不卖 | 标准化产品·卖给加盟商 | 不竞争，各吃各的饭 |
| **星巴克** | 全球统一·云端AI | 边缘盒子·视觉驱动 | 不同品类，不同战场 |
| **海康/大华** | 硬件+通用AI·toG | 火锅专用·toB·做运营 | 用他们的摄像头，做他们不做的 |
