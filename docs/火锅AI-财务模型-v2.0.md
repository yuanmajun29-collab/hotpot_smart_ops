## 六、定价策略建议

### 6.1 推荐定价路径

```
Phase 1: 试点价 (前3-6月，≤10家)
  └─ 方案A试点价 ¥12,000硬件 + ¥999/月
     └─ 目标: 快速积累案例 + ROI数据

Phase 2: 标准价 (第6-12月，冯校长浙江推广)
  └─ 方案A标准价 ¥15,000硬件 + ¥1,999/月
  └─ 方案B ¥500+¥2,499/月（降低进入门槛）
     └─ 目标: 跑通10-20家，验证复购

Phase 3: 分层定价 (Y2+，多客户类型)
  └─ 单店版: ¥1,499/月（减配: 仅后厨损耗+桌态）
  └─ 标准版: ¥1,999/月（标配: +SOP+行为）
  └─ 专业版: ¥2,999/月（全功能: +食材+多店总览）
  └─ 企业版: ¥4,999+/月（50+店批量，含私有部署+定制）
```

### 6.2 价格锚点策略

| 价格锚 | 效果 |
|--------|------|
| 对比海康AI Box ¥20,000+/年 | 火瞳便宜40-60%，且做运营不做监管 |
| 对比1个后厨巡检员 ¥72,000/年 | 火瞳 ¥24,000/年，省¥48,000——"1/3的价格替代1个人" |
| 对比食材年损耗 ¥80,000+/年 | 火瞳年费仅为损耗的25%——"花¥24,000省¥80,000" |
