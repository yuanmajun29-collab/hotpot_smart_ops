## 三、内容创作体系 (Ch4 方法论)

### 3.1 内容矩阵

```
顶层（引流）: 行业洞察 + 对比短视频
中层（信任）: 技术拆解 + 数据报告 + 客户案例
底层（转化）: 产品演示 + ROI计算器 + 免费试用
```

### 3.2 高频选题库

| 类型 | 选题示例 |
|------|---------|
| 痛点共鸣 | "你的火锅店后厨每天浪费多少肉？算完吓一跳" |
| 数据震撼 | "火锅店3大隐形杀手：废料第一、人工第二、食安第三" |
| 技术干货 | "Jetson边缘AI在火锅后厨的落地实录" |
| 客户案例 | "冯校长用了3个月，后厨人工省了2个人" |
| 对比出圈 | "有AI vs 无AI 后厨一天差多少钱？" |
| 行业洞察 | "2026年火锅连锁店必须关注的5个技术趋势" |

### 3.3 AI写作工作流

```
1. Perplexity 查行业数据 + 竞品动态
2. ChatGPT 生成初稿（按AIDA结构）
3. Claude 润色语气（老板能懂的语言）
4. 人工校准（加案例/加数据/加个人态度）
5. 多平台改写（公众号→知乎→抖音脚本）
```

## 四、销售漏斗设计 (Ch10 方法论)

### 4.1 五阶段漏斗

| 阶段 | 目标 | 方法 | 转化率 |
|------|------|------|--------|
| ① 认知 | 让老板知道AI能管后厨 | 短视频/知乎/公众号 | 1000→50 (5%) |
| ② 兴趣 | 让老板想了解详情 | 深度文章+案例+行业报告 | 50→20 (40%) |
| ③ 信任 | 让老板相信你靠谱 | **30天免费试用** + 同行推荐 | 20→10 (50%) |
| ④ 决策 | 让老板下决心付费 | 上门演示 + ROI测算 + 限时优惠 | 10→3 (30%) |
| ⑤ 裂变 | 让付费老板推荐新客户 | 转介绍奖励 + 案例共创 | 3→4.5 (1.5x) |

### 4.2 核心转化策略: 30天零风险试用

**降门槛四件套**:
1. "设备我们出，试用30天，满意再付"
2. 试用期内自动生成对比报告（有AI vs 无AI数据）
3. 30天到期自动发送ROI报告：省了多少钱
4. "不满意随时拆走，零风险"

### 4.3 微信私聊销转三板斧 (Ch10)

| 招数 | 话术示例 |
|------|---------|
| 欲擒故纵 | "冯校长，你上次去后厨是什么时候？" |
| 信息差刺激 | "我最近发现一个方法，能把火锅店原料浪费降20%..." |
| 社会证明 | "冯校长椒江店用了三个月，后厨人工减了2人" |

## 五、社群运营 (Ch8 方法论)

### 5.1 种子用户获取

| 来源 | 策略 |
|------|------|
| 学员群 | 现有技术学员中筛选火锅从业者 |
| 冯校长转介绍 | 标杆客户的同行朋友圈 |
| 公众号粉丝 | 筛选关注餐饮/技术的粉丝 |

### 5.2 周运营SOP

| 日 | 动作 |
|----|------|
| 周一 | 行业热点推送 + 产品周报 |
| 周三 | 客户案例分享 + "老板茶话会"答疑 |
| 周五 | 本周数据亮点 + 下周预告 |

### 5.3 五层粉丝转化

```
粉丝(关注) → 用户(加群) → 成员(试用) → 合伙人(付费) → 传播者(推荐)
```
