## 主持人提纲（两场合用，50min）

| 时段 | 内容 | 对应记录表 |
|------|------|------------|
| 0-5min | 说明目的：听意见、不考核；演示环境说明 | §0 场次信息 |
| 5-10min | 展示首页 → 问「早上最先看哪 3 个数？」 | §1.1 |
| 10-18min | 桌态 + Top5 → 问「待清桌展示够吗？」 | §1.2 |
| 18-28min | PDA 5 步走一遍 → 计时，问能否 3min | §1.3 + UAT-P |
| 28-35min | 告警页 + 企微文案 → 问「会不会嫌烦？」 | §1.4 |
| 35-42min | 日报四段 → 问「缺什么章节？」 | §1.5 |
| 42-48min | UAT 快勾（能勾就勾，不能勾记 BL-xx） | §2 |
| 48-50min | 综合评分 + 店长签字 | §3~5 |
