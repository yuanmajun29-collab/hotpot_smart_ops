## 二、10家逐家对标

### 1. McDonald's — 全球最大AI买家 (43,000店)

| | 详情 |
|------|------|
| **AI投入** | 2019年$300M收购Dynamic Yield(个性化推荐引擎，2021.12已出售给Mastercard) + 收购Apprente(语音AI) |
| **核心能力** | ☁️动态菜单定价 ☁️语音点餐 ☁️供应链预测 |
| **边缘视觉** | ❌ 无 |
| **厨房AI** | ❌ 无。标准化流程靠人 |
| **对外销售** | ❌ 全自用 |
| **单店硬件** | $5-10K (自助Kiosk) |

### 2. Domino's — 最像火瞳的公司 (20,000店)

| | 详情 |
|------|------|
| **核心系统** | 📷 **DOM Pizza Checker** — 摄像头拍披萨→AI判断质量 |
| **AI能力** | 📷披萨出品视觉质检 ☁️需求预测 ☁️AnyWare语音 |
| **边缘视觉** | 🟡 **有！** 但跑在云端，非边缘推理 |
| **对外销售** | ❌ 自用 |
| **单店硬件** | ~$2K (平板+摄像头) |
| **vs火瞳** | DOM只检测1个披萨。火瞳覆盖8个后厨场景。边缘 vs 云端。 |

## 三、核心发现

### 谁在"边缘视觉 × 厨房监控 × 对外销售"的交集？

```
全球餐饮科技矩阵:

              有厨房CV  边缘推理  对外销售  价格/店
Domino's DOM     🟡       ❌       ❌      $2K
Miso Flippy     🟢       🟢       🟢      $20-50K
Amazon JWO      ❌(零售)  🟢       🟢      $500K
Chipotle        🟢       🟢       ❌      $100K
★ 火瞳           🟢       🟢       🟢      ¥7K
────────────────────────────────────────────────
三者全绿的: 只有火瞳和Miso。但价格差5-7倍。
```

### 为什么巨头们"有能力但不做"？

| 巨头 | 技术能力 | 不做火瞳的原因 |
|------|:------:|------|
| McDonald's | 🟢 | 厨房高度标准化→不太需要视觉监控 |
| Domino's | 🟢 | 披萨标准化→只做一个品就够了 |
| Amazon | 🟢 | 零售够大→餐饮厨房不是优先级 |
| 海底捞 | 🟢 | 自用→不对加盟商→不需要产品化 |
| 瑞幸 | 🟢 | 云端数据已够强→不需要边缘视觉 |

### 火瞳的全球独占性

```
全球12家餐饮科技巨头中:
- 10家没有边缘视觉AI
- 1家有但做零售(Amazon JWO)
- 1家有但太贵(Miso $36K/年)
- 0家在火锅后厨场景
- 0家做加盟连锁管控
- 0家做到¥650/月的价格点

→ 火瞳在全球"火锅后厨边缘视觉 × 加盟管控 × 万元定价"
  三个条件全命中，独占。
```

## 四、从全球对标中学到的

| 巨头 | 火瞳应该学的 | 火瞳应该避开 |
|------|------------|------------|
| **Domino's** | AI厨房品控已被验证可行 | 只做一个品(披萨)，不够 |
| **Miso Robotics** | 厨房AI可以卖 | $3K/月太贵，卖不出去 |
| **Amazon JWO** | 边缘视觉天花板验证 | $500K/店太重，规模不了 |
| **Chipotle** | 机器人进厨房可行 | $100K直营专属，加盟商买不起 |
| **瑞幸** | "数据管2万店"是最好的叙事 | 纯云端，没视觉独特性 |
| **McDonald's** | SaaS技术摊薄经济学 | 自用闭环，不会卖 |

## 五、终极定位

> **全球餐饮科技行业里，火瞳是唯一一个：
> 用边缘视觉AI + 万元价格 + 卖给加盟连锁
> 的产品。**

> Domino's有视觉但自用。Amazon有视觉但太贵。Miso有销售但价格离谱。
> 火瞳的三个要素全命中——这是全球空白，不是局部蓝海。
