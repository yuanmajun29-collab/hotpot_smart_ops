# 火瞳 — 平台+边缘计算产品市场调研

> 版本: v2.0 | 日期: 2026-07-16 | 基于技术调研v2 + 竞品v2交叉推导

## 一、"AI盒子+SaaS平台"产品形态分析

### 1.1 形态定义

火瞳的产品形态是双引擎:
```
边缘层: Jetson Orin盒子 (¥4,999硬件)
  └→ YOLO推理 + VLM精识别 + 本地配置管理
平台层: SaaS订阅服务 (¥1,999/月/店)
  └→ 数据聚合 + 趋势分析 + 日报推送 + 多店管理
```

对比竞品形态:

| 产品形态 | 代表 | 硬件模式 | SaaS模式 | 毛利率 |
|------|------|:---:|:---:|:---:|
| **纯SaaS** | 客如云/美团 | 无 | ¥299-999/月 | 70-80% |
| **纯硬件** | 海康摄像头 | 买断¥300-800 | 选配云存储 | 30-40% |
| **硬件+SaaS(安防)** | 海康AI Box | 买断¥2,000-5,000 | 云服务¥99-299/月 | 硬30%+软70% |
| **硬件+SaaS(餐饮AI)** | 火瞳 | 买断¥4,999 | ¥1,999/月 | 硬15%+软80% |

*来源: 技术调研v2硬件价格 + 财务模型v2定价 + 客如云官网*

### 1.2 双引擎模式的优劣

**优势**:
- 硬件锁定: 盒子装了就很难换(迁移成本高)
- 数据飞轮启动快: Edge处理→Hub学习→模型越好→客户越不换
- 单店LTV高: (¥4,999 + ¥1,999×12) = ¥28,987/年/店
- 连锁效应: 总代买1台体验→5台→50台(自然扩张)

**劣势**:
- 初始投入高: 单店¥4,999硬件是决策门槛( vs 纯SaaS¥0)
- 硬件维护成本: 每店需要1天安装+潜在售后
- 供应链风险: Jetson Orin供货波动(技术调研v2: 供货2-4周)
- 毛利率被硬件拉低: 综合毛利≈65%(硬15%+软80%加权)

### 3.3 "硬件+SaaS"模式的毛利率

```
方案A 单店经济模型(稳态, 20+店):
  收入: ¥4,999(硬件) + ¥1,999×12(SaaS) = ¥28,987/年
  硬件成本: ¥4,999 × 85% = ¥4,249
  SaaS成本: ¥1,999 × 12 × 20% = ¥4,798
  总成本: ¥9,047
  毛利: ¥19,940/年/店
  毛利率: 68.8%
```

*注: SaaS成本20%含服务器+带宽(技术调研v2: 阿里云¥92/月 + 带宽¥50/月)*

| 行业 | 公司 | 模式 | 状态 |
|------|------|------|:--:|
| 安防 | 海康/大华/宇视 | NVR+云存储 | ✅ 千亿市场 |

**关键发现**: "硬件+SaaS"模式在安防和农业已充分验证, 餐饮是未开垦的处女地。

### 4.2 火瞳的产品壁垒

```
Level 1: 硬件锁定 — 盒子装了, 换竞品需重新布线+部署+培训 (迁移成本¥3,000+)
Level 2: 数据飞轮 — 用得越久, 火锅专用模型越准, 客户越离不开
Level 3: 网络效应 — 总代一签多店, 区域密度越高, 数据越有价值
```

*来源: 可行性分析v2.0*
