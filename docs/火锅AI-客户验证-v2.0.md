## 一、火锅店成本结构（行业基准推导）

### 1.1 成本拆解（占营收百分比）

基于中国饭店协会/红餐网行业报告数据，火锅店典型成本结构如下：

| 成本科目 | 占比 | 年金额(¥300万店) | 年金额(¥500万店) | 说明 |
|---------|:----:|-----------------|-----------------|------|
| 食材/原料成本 | **35-40%** | ¥105-120万 | ¥175-200万 | 火锅品类食材品类多、损耗大 |
| 人力成本 | **20-25%** | ¥60-75万 | ¥100-125万 | 后厨切配+前厅服务，火锅用工多 |
| 房租物业 | **10-15%** | ¥30-45万 | ¥50-75万 | 商业综合体/街铺，火锅面积≥200㎡ |
| 水电燃气 | **3-5%** | ¥9-15万 | ¥15-25万 | 火锅电磁炉/燃气消耗大 |
| 营销/平台佣金 | **3-5%** | ¥9-15万 | ¥15-25万 | 美团/大众点评抽佣+推广 |
| 其他(折旧/耗材/杂费) | **5-8%** | ¥15-24万 | ¥25-40万 | 锅底/蘸料/一次性用品等 |
| **净利润** | **8-15%** | ¥24-45万 | ¥40-75万 | 存量竞争期利润被压缩 |

> 数据来源: 中国饭店协会《2024中国餐饮业年度报告》+ Frost & Sullivan火锅行业成本结构模型

### 1.2 存量竞争下的利润侵蚀

| 指标 | 2019(增量期) | 2024(存量期) | 变化 |
|------|:-----------:|:-----------:|:----:|
| 净利润率 | 12-18% | **8-15%** | ↓3-6pp |
| 食材成本占比 | 33-38% | 35-40% | ↑2pp |
| 人力成本占比 | 18-22% | 20-25% | ↑2pp |
| 房租占比 | 10-13% | 10-15% | ↑2pp |

**推论**: 存量市场按年6175亿规模计算，每提升1%净利润 = 行业多出**61.75亿**利润空间。降本工具的价值被放大。

## 三、翻台率行业基准

### 3.1 火锅翻台率数据

| 指标 | 数值 | 来源/说明 |
|------|:----:|------|
| 火锅行业平均翻台率 | **1.5-2.0次/天** | 低于正餐(2.5-3.0)，因就餐时间长(60-90分钟/桌) |
| 海底捞翻台率 | 3.0-4.0(2023) → 3.5(2024) | 行业标杆，上市披露 |
| 呷哺呷哺翻台率 | 2.4-2.8 | 小火锅翻台更快 |
| 中小火锅店翻台率 | 1.2-1.8 | 峰谷明显，午市空置率高 |
| 每提升0.1翻台率 | +¥15-25万/年(¥300万店) | 每桌消费¥200-400 × 2-4人 |

### 3.2 翻台率瓶颈分析

| 瓶颈 | 影响 | 火瞳解决方案 |
|------|------|------------|
| 桌态识别滞后（空桌不知） | 每桌浪费5-15分钟 | 前厅plan_a/plan_b实时桌态→推送 |
| 加汤不及时（顾客等待） | 就餐时间延长10-20% | 加汤时机检测→提醒 |
| 空盘未撤收（占桌不翻） | 空桌30-60分钟才翻 | 空盘撤收提醒→服务员 |
| 后厨出菜慢（前厅等菜） | 平均延长15-25分钟 | 后厨SOP→出菜效率监控 |

**火瞳翻台提升预估**: 每店提升0.15-0.30次/天 → **年增收¥22,500-75,000**（按¥300万店，每0.1翻台¥15-25万收入）

## 四、三层客户画像（深化版）

### 4.1 画像一：单店老板「老王」

| 维度 | 详情 |
|------|------|
| **典型画像** | 45岁，自有1-2家火锅店(川渝火锅)，自己盯店 |
| **门店营收** | ¥200-500万/年，净利润¥16-75万 |
| **后厨规模** | 3-6人（主厨1 + 切配2-3 + 洗碗1-2） |
| **日常痛点** | "我不在店里就不放心"——后厨有没有偷懒？浪费了多少？有没有以次充好？ |
| **管理现状** | 靠人盯、靠信任、靠月报（滞后2-4周） |
| **决策心理** | 感性驱动："朋友推荐 + 听起来有用" → 理性验证："省多少钱？回本多久？" |
| **决策周期** | **1-4周**，熟人推荐可压缩到1周 |
| **预算上限** | 月费 > ¥2,000会犹豫，年总支出 < ¥2万 |
| **触发词** | "帮你省一个厨师的工资" "你不在店里也能看到后厨" |
| **获客方式** | 同行推荐 > 餐饮协会 > 火锅展会 |
| **年损失暴露** | 后厨损耗 ¥15-26万 + 巡检人工 ¥7.2万 = **¥22-33万** |
| **火瞳ROI** | 投入¥2万 → 年省¥11-17万 → ROI **550-850%** |
| **转化率预估** | 试用后转化 40-60%（¥2,000/月以下定价） |
| **风险** | 决策感性，若早期案例口碑不好会迅速扩散 |

### 4.2 画像二：区域总代「陈总」

| 维度 | 详情 |
|------|------|
| **典型画像** | 38岁，手握某火锅品牌浙江区域总代权，管理3-15家加盟店 |
| **门店营收** | 单店 ¥300-600万/年，总代抽成/管理费 |
| **后厨规模** | 4-8人/店 × N家店 |
| **核心痛点** | "多家店管不过来，SOP难统一，每家店损耗情况完全不知道" |
| **管理现状** | 每月巡店1-2次，靠店长汇报，数据滞后且可能粉饰 |
| **决策心理** | 理性为主：先试点1家→3个月看数据→全部门店推广。有推广动力（他管这些店） |
| **决策周期** | **2-8周**（含1个月试点期） |
| **预算上限** | 月费 < ¥3,000/店可接受，看多店打包折扣 |
| **关键指标** | 单店ROI > 多店对比Dashboard > 品牌合规率 |
| **触发词** | "所有店后厨损耗一目了然" "哪家店SOP违规最多" "堵住加盟店的进货漏洞" |
| **获客方式** | 品牌体系内推（总代→二级加盟商） > 餐饮展会 > 行业协会 |
| **年损失暴露** | 单店损耗 ¥15-26万 × N + 多店管控盲区隐性损失 = **¥45-390万** |
| **火瞳ROI** | 投入¥2-3万/店 × 3-15店 → 年省¥34-52万(3店)/¥170-258万(15店) |
| **转化率预估** | 试点后转化 60-80%（他有推广权+管理痛点最强） |
| **战略价值** | ⭐⭐⭐⭐⭐ **最佳切入点** — 总代有推广权+管理痛点+决策快 |

### 4.3 画像三：连锁总部「李总」

| 维度 | 详情 |
|------|------|
| **典型画像** | 50岁，火锅连锁品牌运营总监/IT负责人，品牌50-200+门店 |
| **门店营收** | 单店 ¥500-2,000万/年，品牌整体 ¥2.5-40亿 |
| **后厨规模** | 5-10人/店 × 50+店 |
| **核心痛点** | "品牌一致性 + 食品安全风险 + 总部成本管控"——单店损耗可能无感，但50店汇总=巨款 |
| **管理现状** | 已有ERP/收银/SaaS，但缺后厨实时数据，靠季度审计+神秘顾客 |
| **决策心理** | 组织决策：运营部提案→IT评估→采购比价→高层审批→POC试点→签约 |
| **决策周期** | **3-6个月**（需POC+招标+供应商资质审核） |
| **预算上限** | 高（年度IT预算），但要合规+有案例+供应商有资质 |
| **关键指标** | 食品安全合规率 > 成本管控(ROI) > 品牌标准化 > 供应商稳定性 |
| **触发词** | "全国门店后厨数据一张图" "食品安全0事故" "每店每年节流XX万×50店" |
| **获客方式** | BD直接拜访 > 餐饮展会主题演讲 > 行业报告/白皮书 > 标杆案例 |
| **年损失暴露** | 单店损耗 ¥15-26万 × 50-200店 = **¥750-5,200万** |
| **火瞳ROI** | 投入¥1.5-2万/店 × 50-200店 → 年省¥400-2,600万 |
| **转化率预估** | POC后转化 30-50%（组织决策慢但粘性高，续费率>90%） |
| **风险** | 决策链长，易卡在IT评估/采购环节；需公司资质+标杆案例；一旦签下是长期客户 |
| **先决条件** | 需先有10+店案例+冯校长标杆+产品成熟度5/5 |

## 五、客户分层获客策略

```
    获客难度
       ↑
    连锁总部  │  ⚠️ 长期目标 (3-6月决策)
    (50+店)   │     需: BD + 案例 + 资质
              │
    区域总代  │  🎯 核心突破口 (2-8周决策)
    (3-15店)  │     需: 试点 + ROI数据
              │
    单店老板  │  ✅ 快速验证 (1-4周决策)
    (1-2店)   │     需: 熟人推荐 + 免费试用
              │
              └──────────────────→ 客户价值(LTV)

    单店: LTV ¥2万/年 × 3-5年 = ¥6-10万
    总代: LTV ¥2-3万/年 × 3-5年 × 3-15店 = ¥18-225万
    总部: LTV ¥1.5-2万/年 × 5-10年 × 50-200店 = ¥375-4,000万
```

### 核心策略

1. **先打总代**: 决策快(2-8周)+ 自有推广动力 + 一签多店
2. **单店做案例**: 熟人推荐破冰，快速积累口碑
3. **总部做远期**: 需5-10个成功案例后启动BD
