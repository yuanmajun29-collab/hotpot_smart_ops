# 火瞳 — 开发计划

> 3阶段/12个月 | 基于现有代码重构+新增

---

## 一、里程碑总览

```
Phase 1: 试点验证（1店 · 2个月）
  └→ Phase 2: 冯校长全门店（3-10店 · 6个月）
       └→ Phase 3: 标品全国（50+店 · 12个月）
```

| Dashboard试点版精简 | 1天 | P0 |

### Week 3-4: 数据采集 + 调试

| 任务 | 内容 | 优先级 |
|------|------|:--:|
| 后厨数据采集 | 7天×24h原始数据积累 | P0 |
